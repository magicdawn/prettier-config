## Install

```sh
$ cnpm i -D @magicdawn/prettier-config
```

## Usage

### `package.json`

```json
{
  "prettier": "@magicdawn/prettier-config"
}
```

### `prettier.config.js`

```js
module.exports = {
  ...require('@magicdawn/prettier-config'),
}
```
